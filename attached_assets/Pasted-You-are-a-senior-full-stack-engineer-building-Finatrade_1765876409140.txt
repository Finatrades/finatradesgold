APPROVAL LOGIC (ATOMIC)
When admin approves txn:
1) Load txn (must be PENDING)
2) Fetch price_per_gram from metal-api
3) Compute grams = round(usdAmount / price_per_gram, 4)
4) Update transaction:
   - status = APPROVED
   - goldGrams = grams
   - approvedAt = now
5) Update FinaPay wallet:
   - usdValue += usdAmount
   - goldGrams += grams
6) Update FinaVault ledger:
   - add CREDIT entry with grams, usdReference, statusAfter=AVAILABLE, source=EXTERNAL, dest=FINAPAY
7) Update Vault wallet view:
   - usdReference += usdAmount
   - availableGrams += grams
8) Write transaction history view (from transactions table)
All steps must be done in a DB transaction.
